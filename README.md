# express_template
基于 express 的接口开发
~~做成基于express 的后台接口开发框架模板~~

### 使用

```shell

安装依赖
yarn install

运行
node app

```

### 项目目录结构

```
├── app.js             # 启动文件
├── src               
│   ├── config         # 基础配置
│   ├── controllers    # 控制层
│   ├── docs           # 接口文档
│   ├── middlewares    # 中间件
│   ├── models         #
│   ├── routes         # 路由配置目录
│   ├── utils          # 工具函数目录
│   └── index.js       # 启动文件
```
